# Web Kilat Ultimate Engine (V3)
Standardize WhatsApp Cart Engine for Perintis Digital Landing Pages.

## Required HTML IDs
To use this engine, ensure your HTML contains these IDs:

| ID | Type | Description |
|---|---|---|
| `cart-drawer` | div | The main cart sliding container |
| `cart-items` | div | Container where items are injected |
| `cart-total` | span/div | Where the total RM price is shown |
| `cart-badge` | span | Counter in Navbar |
| `cart-badge-floating` | span | Counter in Floating Button |
| `cust-name` | input | Customer Name field |
| `cust-phone` | input | Customer Phone field |
| `cust-address` | textarea | Customer Address field |
| `checkout-whatsapp` | a | Trigger for WhatsApp checkout |
| `copy-order` | a | Trigger for Copy to Clipboard |
| `cart-toggle` | a | Navbar cart opener |
| `cart-floating-toggle`| a | Floating cart opener |

## Usage
Add this line before `</body>`:
`<script src="https://cdn.jsdelivr.net/gh/USERNAME/REPO_NAME/wk-engine-v3.js"></script>`
